# Brave Hearts Publishing Teacher Resources Page

## WordPress assignment

1. Create or edit the WordPress page named Teacher Resources.
2. Use the slug teachers.
3. Select Brave Hearts Teacher Resources Page in the page-template selector.
4. Publish or update the page.
5. Confirm header, footer, Books page, and About page links point to /teachers/.

Because the file is named page-teachers.php, WordPress can select it automatically for the teachers slug.

## Resource availability

All six resource category cards default to Resource coming soon. A card becomes available only when its corresponding public custom field contains an approved URL:

- bhp_teachers_lesson_plans_url
- bhp_teachers_discussion_guides_url
- bhp_teachers_vocabulary_url
- bhp_teachers_maps_url
- bhp_teachers_printables_url
- bhp_teachers_read_aloud_resources_url

Do not add a URL until the destination or file exists and has been reviewed. No fake downloadable files are generated by the template.

## Book-based resources

The page uses the same three adventure records as the Books page:

- The Mariana Trench
- Mount Everest
- The Amazon Rainforest

WooCommerce product links and covers come from bhp_get_series_adventures. Classroom themes and suggested subjects are defined in the Teacher Resources template.

Book-specific classroom resource links use:

- bhp_teachers_mariana_trench_resources_url
- bhp_teachers_mount_everest_resources_url
- bhp_teachers_amazon_rainforest_resources_url

Without an approved resource URL, the card displays Coming Resources as a disabled state.

## Read-aloud inquiries

The default Request a Read-Aloud link points to the Contact page with an inquiry value of read-aloud. Override it with:

- bhp_teachers_read_aloud_url

Before launch, confirm the inquiry destination explains availability, location, schedule, audience size, school requirements, and response expectations.

## Teacher signup handling

The page uses the Phase 5A teacher-resource-signup component.

Required segmentation:

- audience_type: teachers
- lead_magnet: teacher_resources
- source_page: the Teacher Resources page permalink

The teacher_resources lead magnet is registered as a placeholder. Email input and submission remain disabled until a real action is supplied through the provider-neutral bhp_signup_form_action filter. No Mailchimp integration or fake submission is included.

## Optional hero media

Set bhp_teachers_hero_image_id to a valid WordPress attachment ID to add an approved classroom or adventure image to the hero.

## Manual setup before launch

- Create the Teacher Resources page and assign the template.
- Verify every book cover and WooCommerce link.
- Add only approved resource URLs.
- Configure and test the read-aloud inquiry route.
- Approve teacher signup copy and consent language.
- Connect the provider endpoint in a later integration phase.
- Test mobile, tablet, keyboard, and screen-reader behavior in live WordPress.
